Como foi feita a ferramenta de acessibilidade

vamos começar por, o que é a ferramenta, a ferramenta foi feita com a simples intenção de fazer com que todas as pessoas com daltonismo pudessem ver o site da mesma maneira que as outras, porém, como a ideia principal é algo praticamente impossivel de se fazer, tivemos que fazer algo diferente pra alcançar esse objetivo

O grey scale

O grey scale, ou escala de cinza, foi o método que decidimos usar pra alcançar o objetivo principal, uma ferramenta de acessibilidade que faz com que seja possível ver o site inteiro em tons de cinza, e você me pergunta "pra que?" e eu te respondo, porque isso tira basicamente toda a dificuldade de pessoas com qualquer deficiência de cor para enxergar o site de uma mesma maneira

Mas por que o grey scale?

Simples, porque o greyscale executa a função que eu precisava que ele executasse de maneira perfeita e simples, eu queria que fosse diferente, mas daria trabalho demais, ja que eu teria que fazer uma ferramenta do zero, que fizesse uma função extremamente complexa, talvez quando eu tiver com mais tempo eu consiga fazer o objetivo principal, mas no momento, com o tempo limitado, não deu, então o grey scale foi a melhor maneira pra fazer essa função com o tempo que eu tinha

E agora, vamos pra explicação do código

Primeiro o código no css

html{ padding: 0; margin: 0; --blue: #0000ff; --darkerBlue: #0000c8; --white: #ffffff; } que dentro da sessão do html, eu criei as variaveis de cor, que são extremamente importantes, entãoo, se quiser usar a função do greyscale, eu recomendo que faça isso se não quiser passar muito trabalho

E agora o código do java script, que por ser um pouco mais complexo eu vou fazer comentários dentro do próprio código pra explicar

const html = document.querySelector("html") const checkbox = document.querySelector("input[name=theme]") const getStyle = (elemente, style) => window.getComputedStyle(html).getPropertyValue("--bg") const initialColors = { blue: getStyle(html, "--blue"), darkBlue: getStyle(html, "--darkBlue"), white: getStyle(html, "--white"), } const greyScale = { blue: "#ffff00", darkBlue: "ffff37", white: "#000000", } const transformKey = key => "--" + key.replace(/([A-Z])/, "-$1").toLowerCase() const changeColors = (colors) =>{ Object.keys(colors).map(key => html.style.setProperty(transformKey(key), colors[key]) ) } checkbox.addEventListener("change", ({target}) =>{ target.checked ? changeColors(greyScale) : changeColors(initialColors) })
